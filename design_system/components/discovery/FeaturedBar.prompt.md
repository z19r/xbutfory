Full-bleed "AS FEATURED ON" social-proof strip — glossy gel badges and a retro `valid XHTML 1.0 · RSS 2.0` stamp. Place between the feed and footer.

```jsx
<FeaturedBar badges={['Digg', 'del.icio.us', 'MetaFilter', 'Daring Fireball']} />
```

Props: `label`, `badges` (string[]), `note` (pass '' to hide the validity stamp), `gel` (glossy badges, default on), `maxWidth` (align with page grid). Full-bleed band — render it edge-to-edge, not inside a centered column.

Drops a monospace "code chip" into running serif copy — the house treatment for the literal phrase `X but for Y`.

```jsx
<h1 style={{ fontFamily: "var(--font-display)" }}>
  …pitch themselves as <CodeChip>X but for Y</CodeChip>.
</h1>
```

- Sized in `em`, so it scales with the headline it sits in. Use sparingly — it's a punctuation mark, not a button.
